Water Level Indicator

  • Project Name: Water Level Indicator
  • Description: A device to monitor and indicate water levels in a tank to prevent overflow and dry running of pumps.
  • Key Features:
    • LEDs indicate Low, Medium, and High water levels
    • Buzzer alerts when tank is full
    • Uses metal probes or sensors for level detection
  • Components Used:
    • Transistors
    • LEDs
    • Resistors
    • Buzzer
    • Wires
    • Power Supply
  • Applications: Home water tanks, overhead tanks, industrial water storage systems
  • Technology: Basic electronics and water conductivity sensing

To-Do List CLI App

The To-Do List Command-Line Interface (CLI) App is a Python-based productivity tool designed to help users manage their tasks effectively. Key features include:

  • Task Management: Add new tasks, mark tasks as completed, and delete tasks.
  • Persistence: Tasks are saved using file handling or Python’s dictionary structure, ensuring data isn’t lost after the program exits.
  • User-Friendly Commands: Simple and intuitive commands like add, delete, view, and complete make it easy to navigate and use.
  • Organized Output: The app displays tasks in a well-structured format, highlighting completed ones for better visualization. This project introduces users to basic Python concepts like file handling, conditionals, and loops while solving real-world problems.

Tech: Python, File I/O

Number Guessing Game

The Number Guessing Game is a fun and interactive Python project that challenges players to guess a randomly generated number. Features include:

  • Dynamic Range: The game generates numbers within a specified range, adjustable for different difficulty levels.
  • Hints: After each guess, players receive feedback, such as “too high” or “too low,” helping them refine their guesses.
  • Limited Attempts: To add challenge, players are given a set number of attempts, with an option to play again if they fail.
  • Performance Summary: At the end of the game, a summary of the player’s success rate is displayed. This project highlights Python’s random module, conditional statements, and loop constructs while providing entertainment through gameplay.

Tech: Python, Random Module
